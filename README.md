# Projet Awale - IA Game Programming

Implémentation d'une IA compétitive pour le jeu d'Awale avec plusieurs niveaux d'évaluation.

## Structure

- **Arene/** : Arbitre et protocole de tournoi
- **version-finale/** : Version optimale pour tournoi

## Compilation

```bash
Sous linux : 

Tout se passe dans le dossier Arene/ :
make clean 
make
Puis mettre le nom du fichier super_bot ainsi que celui de l\'adversaire dans le fichier Arbitre.java 

# Arene
javac Arbitre.java
java Arbitre

```
